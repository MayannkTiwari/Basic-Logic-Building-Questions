# Basic Logic Building Questions in Java

This repository contains beginner-friendly Java programs focused on improving logical thinking, problem-solving skills, and programming fundamentals.

The goal of this repository is to build a strong foundation in Java by solving basic logic-building questions and gradually progressing to more challenging problems.

## How to Run

1. Clone the repository

```bash
git clone https://github.com/MayanknTiwari/Basic-Logic-Building-Questions.git
```

2. Navigate to the project folder

```bash
cd Basic-Logic-Building-Questions
```

3. Compile a Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```
